# Tic Tac Toe

A simple **Tic Tac Toe** web game built with **JavaScript, HTML, and Tailwind CSS**.  
The project focuses on practicing **JavaScript modules, factory functions, and DOM manipulation** while keeping global code minimal.

---

## Features

- Two-player Tic Tac Toe game  
- Players can enter their **custom names**  
- **Start / Restart** button to reset the game  
- **Win detection** for all possible combinations  
- **Tie detection** when the board is full  
- **Clickable board interface**  
- Result display showing the **winner or a tie**

---

## Technologies Used

- **HTML**  
- **JavaScript (ES6)**  
- **Factory Functions**  
- **Module Pattern (IIFE)**  
- **DOM Manipulation**  
- **Tailwind CSS**

---

## How to Play

1. Enter names for Player 1 and Player 2 (optional).  
2. Click **Start / Restart Game**.  
3. Click a cell on the board to place your mark.  
4. Players alternate turns automatically.  
5. The game ends when:  
   - A player gets **three in a row**, or  
   - The board is **full (tie)**.
